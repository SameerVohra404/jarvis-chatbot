# Jarvis Chat Agent

A simple Python-based chat agent built using the OpenRouter API with the DeepSeek R1 (free) model.  
This project was created to practice API usage, prompt handling, and basic chatbot structure in Python.

## Features
- Uses OpenRouter's API endpoint for LLM responses  
- Stores conversation history in memory  
- Simple and beginner-friendly structure  
- Clean code with no external dependencies other than the OpenAI client

## Requirements
- Python 3.8+
- `openai` Python package
